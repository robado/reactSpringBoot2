### React.js frontend II

Run application:

#### 1. Install webpack 
(This is what worked and works with the newest webpack)
    
    npm install --save-dev webpack-cli
    
#### 2. Run webpack
    
    webpack
or
    
    webpack-cli
    
#### 3. Run application

    ./mvnw spring-boot:run
    mvnw spring-boot:run
